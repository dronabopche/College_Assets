# Python Programs – How to Execute

## Prerequisites
Python 3 must be installed:
```bash
python3 --version
# If not installed:
sudo apt install python3
```

---

---

## How to Run Each Program

```bash
# 22 – Add Two Numbers
python3 22_add_two_numbers.py

# 23 – Average of Five Numbers
python3 23_average_five_numbers.py

# 24 – Sum up to N
python3 24_sum_upto_n.py

# 25 – Factorial
python3 25_factorial.py

# 26 – Fibonacci Series
python3 26_fibonacci.py

# 27 – Prime Number
python3 27_prime.py

# 28 – Maximum in a List
python3 28_maximum_in_list.py
# Input example: 5 3 8 1 9 2

# 29 – Bubble Sort
python3 29_bubble_sort.py
# Input example: 64 34 25 12 22

# 30 – Searching in a List
python3 30_searching_in_list.py
# Input example list: 10 20 30 40 50
# Then enter the key to search

# 31 – Sum of List Elements
python3 31_sum_of_list.py
# Input example: 1 2 3 4 5

# 32 – Matrix Multiplication (2x2 hardcoded)
python3 32_matrix_multiplication.py
```

---

## Notes
- Programs 28–31 accept space-separated numbers as input.
- Program 32 uses hardcoded matrices `A = [[1,2],[3,4]]` and `B = [[5,6],[7,8]]`. Edit the file to change values.
- On some systems use `python` instead of `python3`.
